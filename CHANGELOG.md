# Changelog

All notable changes to the Cybersecurity Toolkit will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [3.0.0] - 2025-01-01

### Added
- Comprehensive cybersecurity suite with multiple integrated tools
- Network security analysis (UFW, port checking)
- Firewall management (fban2)
- Docker container exposure detection
- File system security scanning
- Interactive mode for user-friendly operation
- Multiple output formats (tree, JSON, Markdown, plain text)
- Security-focused directory mapping functionality
- Both Python and Shell implementations of all tools

### Changed
- Complete rewrite of the security tools with modular architecture
- Improved documentation and user guides
- Enhanced error handling and logging
- Better integration between different security modules

### Fixed
- Various security vulnerabilities in file handling
- Improved dependency checking and validation

## [2.0.0] - 2024-06-01

### Added
- Directory mapping with security analysis features
- Support for multiple output formats
- Integration with .mapignore files
- Detection of sensitive files and weak permissions

## [1.0.0] - 2024-01-01

### Added
- Basic directory mapping functionality
- Tree format output
- Command-line interface
